# Project Images

Place your project images here. The template currently uses SVG placeholders (`project-1.svg`, `project-2.svg`, `project-3.svg`).

## Replacing Placeholder Images

1. Add your project images (JPG, PNG, or WebP format)
2. Name them: `project-1.jpg`, `project-2.jpg`, `project-3.jpg` (or match your naming in `js/main.js`)
3. Update the `image` property in the `PROJECTS` array in `js/main.js`

## Image Recommendations

- **Format**: JPG (for photos), PNG (for graphics with transparency), or WebP (for best performance)
- **Dimensions**: 1200x800px (3:2 ratio) for best results
- **File size**: Keep under 200KB per image for optimal performance
- **Aspect ratio**: Maintain 3:2 ratio for consistent card layouts

## Example

If you have a project image called `my-project.png`, update in `js/main.js`:

```javascript
{
    id: "project-1",
    image: "assets/my-project.png",
    // ... rest of project data
}
```
